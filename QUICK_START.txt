🖥️  WINDOWS SETUP:

  1. Open:  start-ai-server.bat  (in Notepad)
  
  2. Find:  set OPENAI_API_KEY=YOUR_OPENAI_API_KEY_HERE
  
  3. Replace YOUR_OPENAI_API_KEY_HERE with your actual key
  
  4. Save and run:  start-ai-server.bat
  
  5. In another terminal:  npm run dev

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🍎 MAC/LINUX/GIT BASH SETUP:

  1. Open:  start-ai-server.sh  (in text editor)
  
  2. Find:  export OPENAI_API_KEY="YOUR_OPENAI_API_KEY_HERE"
  
  3. Replace YOUR_OPENAI_API_KEY_HERE with your actual key
  
  4. Save and run:  bash start-ai-server.sh
  
  5. In another terminal:  npm run dev

🆘 TROUBLESHOOTING:

  "insufficient_quota"
    → Add billing at https://platform.openai.com/account/billing
  
  "Connection Error"
    → Start AI proxy: start-ai-server.bat or bash start-ai-server.sh
  
  "Port 4001 in use"
    → Kill process: netstat -ano | findstr :4001
    → Then: taskkill /PID <PID> /F
  
  "OPENAI_API_KEY not configured"
    → Edit startup script and add your key
